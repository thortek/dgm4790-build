How to use Dean Radar
The Events tab is where to view upcoming events that other department chairs have submitted.
Here you can filter through events by the Date, Department, or Event Name.
Tapping on an event will bring you to the Event Detail section.
If you need to delete the event you're viewing, tap on the Delete button, then either hit Delete or Cancel to confirm.
The Add Event tab will allow you to create an event for a specific department.
